Game Projects

VR Bowling Game

This game is designed in Unity3D. Player uses HTC vive controller to grab and throw the ball. The whole experience is realist comparing to the actual bowling. Making people practice doing bowling without going out.
I implemented VR controller interaction, balls and pins physics and animation.

Flight Simulator

This game is an experiment to apply airplane physics in the game. The plane follow the true physics, it has drag, lift and torque applied to its body

Dusk of summoners

The game built with unreal engine 4. It is multiplayer game that each player where the player kill the other wins. Player could use magic to attack directly with fireball or laser directly, or summon the dwarfs which has long range and close range type. Summoning and attack will both consume magic. The dwarfs AI are implemented with state machine.

OpenGL Roller Coaster

The roller coaster is implemented with spineline and texture mapping.

OpenGL Terrain Generator

Using openGL, texture, and height map to create 3D terrain.

Parallax Mapping

This is implemented in DirectX11. It uses texture map, normal map and height map to create bumpy feeling. The whole program only takes in 4 vertexes, but create a effect of composing thousands of vertexes and triangles.
